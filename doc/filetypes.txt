Shark3D Snake Binary
====================
.cdr - shark3d_snake_binary, used in 'data/scripts/' and 'data/interactiontables/'
.sgr - shark3d_snake_binary, used for shaders, contained in directory 'art/shaders/prog/...'
.sdr - shark3d_snake_binary, used for shaders, contained in directory 'art/locations/.../shader'
.skr - shark3d_snake_binary, shark3d_std_bone_skel_runtime
.sir - shark3d_snake_binary, probally 3d script? see format, shark3d_std_inst_3x0
.svr - shark3d_snake_binary, shark3d_runtime_model_pvs, seldomly used
.ikr - shark3d_snake_binary, tlj2_ikr_0
.rgr - shark3d_snake_binary, funcom_bone_rig_runtime
.spr - shark3d_snake_binary

Multimedia Data:
================
.mp3 - Standard MP3 files
.wav - Standard Wav files
.png - DirectX texture files (ie. not real .pngs), can also be STFU4 file (?)
.tga - DirectX texture files (ie. not real .tgas)
.dds - DirectX texture files
